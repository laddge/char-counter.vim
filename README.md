# char-counter.vim
Minimal Char Counter for vim

## Installation
Vundle:

```
Plugin 'laddge/char-counter.vim'
```

According to the plugin manager you use.

## Usage

The number of characters is assigned to the following variables:

```
b:charCount
```

For statusline, use this:

```
%{b:charCount}
```
